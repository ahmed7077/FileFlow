# DSA File Organiser

##  Overview
DSA File Organiser is a Java-based project that demonstrates the use of Data Structures and Algorithms for managing and organizing files efficiently.

##  Features
- File categorization using custom logic
- Organized module-based architecture
- Uses DSA concepts like sorting, queues, and trees
- Modular design for easy scalability
